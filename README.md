# StorViz

[繁體中文](README.md) | [English](README.en.md) | [日本語](README.ja.md)

現代化儲存空間視覺化分析工具。

## 功能特色

- **視覺化儲存分析**：使用互動式圖表直觀地呈現磁碟空間使用情況
- **快速掃描**：使用 Rust 後端搭配平行處理技術，提供高效能的目錄掃描
- **檔案類型統計**：詳細的檔案類型和副檔名儲存空間分析
- **多語言支援**：支援英文、繁體中文和日文
- **深色模式**：美觀的深色/淺色主題切換
- **跨平台**：可在 macOS 和 Windows 上運行
